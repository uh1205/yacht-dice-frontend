# 🎲 요트 다이스 (Yacht Dice)

> 웹 브라우저에서 플레이 가능한 요트 다이스 게임입니다.  
> 플레이어는 각 턴마다 5개의 주사위를 굴려서 점수를 기록하고, 최종 점수를 경쟁합니다.

---

## 기능 목록

### 1. 주사위 관련 기능

- 5개의 주사위가 존재한다.
- 각 주사위를 개별적으로 고정 또는 해제 할 수 있다.
- 굴리기 버튼을 눌러 고정되지 않은 주사위를 굴릴 수 있다.
- 굴려진 주사위는 각각 1 ~ 6 사이의 랜덤 값을 생성한다.
- 한 턴에 최대 3회까지 주사위를 굴릴 수 있다.

---

### 2. 점수판 관련 기능

- 점수판은 12개의 족보로 구성된다.
- 주사위를 굴린 후, 플레이어는 원하는 족보를 선택하여 점수를 기록할 수 있다.
- 각 족보는 한 번만 선택할 수 있다.
- 모든 족보가 채워지면 게임이 종료된다.

---

### 3. 점수 계산 기능

- 에이스: 1이 나온 주사위 눈의 총합
- 듀스: 2가 나온 주사위 눈의 총합
- 트레이: 3이 나온 주사위 눈의 총합
- 포: 4가 나온 주사위 눈의 총합
- 파이브: 5가 나온 주사위 눈의 총합
- 식스: 6이 나온 주사위 눈의 총합
- 보너스: 에이스 ~ 식스 점수 합계가 63점 이상일 때, 35점을 추가로 얻는다.
- 초이스: 주사위 눈 5개의 총합
- 포 다이스: 동일한 주사위 눈이 4개 이상일 때, 주사위 눈 5개의 총합
- 풀 하우스: 주사위를 2개, 3개로 묶었을 때 각 묶음 안 주사위 눈이 동일할 때, 주사위 눈 5개의 총합
- S. 스트레이트: 이어지는 주사위 눈이 4개 이상일 때, 15점을 얻는다.
- L. 스트레이트: 이어지는 주사위 눈이 5개일 때, 30점을 얻는다.
- 요트: 동일한 주사위 눈이 5개일 때, 50점을 얻는다.
