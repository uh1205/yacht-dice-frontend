# 🎲 요트 다이스 (Yacht Dice)

> 웹 브라우저에서 즐길 수 있는 요트 다이스 게임입니다.
> 플레이어는 각 턴마다 주사위를 굴려 점수를 기록하고, 최종 점수를 경쟁합니다.

<br>

---

## 주요 기능

### 오프라인 플레이 기능

#### 주사위 기능

- 5개의 주사위가 존재한다.
- 주사위를 굴리면 1 ~ 6 사이 랜덤 값을 가진다.
- 특정 주사위를 굴리지 않도록 고정할 수 있다.
- 고정된 주사위를 해제할 수 있다.

#### 점수 계산 기능

- 주사위 결과에 대해 각 족보별 점수를 계산한다.
- 점수는 아래에 명시된 규칙에 따라 계산한다.
- 소계(에이스 ~ 식스 점수 합계)를 계산한다.
- 총합 득점(모든 족보 점수 + 보너스 점수)을 계산한다.

#### 점수판 기능

- 주사위를 굴린 후, 플레이어는 원하는 족보를 선택해 점수를 기록한다.
- 각 족보는 한 번만 선택할 수 있으며, 선택 후 변경할 수 없다.
- 모든 족보가 채워지면 그 플레이어의 게임이 종료된다.

#### 게임 결과 출력 기능

- 모든 플레이어가 점수판을 완성하면 게임이 종료된다.
- 최종 점수의 총합이 가장 높은 플레이어가 우승한다.
- 가장 높은 점수가 여러 명이라면 무승부 처리된다.

### 온라인 플레이 기능

#### 웹 소켓 통신 기능

- 플레이어마다 서버가 발행하는 메시지를 구독한다.
- 게임 방마다 서버가 발행하는 메시지를 구독한다.
- 서버로 메시지를 발행한다.

#### 매칭 기능

- 닉네임을 입력하여 서버로부터 생성된 플레이어를 받는다.
- 매칭 / 매칭 취소 요청을 보낸다.
- 매칭 시 매칭 팝업을 띄운 뒤 게임 화면을 렌더링한다.
- 로컬 스토리지에 플레이어 정보와 매칭된 게임 방 정보를 저장한다.

#### 게임 화면 렌더링 기능

- 웹 소켓으로 받아온 게임 상태를 기반으로 게임 화면을 렌더링한다.
- 뒤로가기 후 돌아오기 / 새로고침 시에도 게임 상태를 다시 받아온다.

## 족보별 점수 계산 규칙

### 상단 항목 (UPPER)

| 이름   | 설명                                                        | 예시             |
| ------ | ----------------------------------------------------------- | ---------------- |
| 에이스 | 1이 나온 주사위 눈의 총합. 최대 5점.                        | 1,1,1,5,6 = 3점  |
| 듀스   | 2가 나온 주사위 눈의 총합. 최대 10점.                       | 2,2,2,5,6 = 6점  |
| 트레이 | 3이 나온 주사위 눈의 총합. 최대 15점.                       | 3,3,3,5,6 = 9점  |
| 포     | 4이 나온 주사위 눈의 총합. 최대 20점.                       | 4,4,4,5,6 = 12점 |
| 파이브 | 5가 나온 주사위 눈의 총합. 최대 25점.                       | 2,3,5,5,5 = 15점 |
| 식스   | 6이 나온 주사위 눈의 총합. 최대 30점.                       | 2,3,6,6,6 = 18점 |
| 보너스 | 상단 항목 점수 합계가 63점 이상일 때, 35점을 추가로 얻는다. | -                |
